# REFLECTION

This project challenged me to design a system that balances deterministic logic with AI-assisted enhancements.

The key learning was that financial recommendation systems must remain predictable and explainable. I intentionally avoided using LLMs for core pricing logic to prevent hallucinations and inconsistencies.

Instead, I built a rule-based audit engine that ensures reproducible outcomes while using AI only for natural language summarization.

Working with Supabase improved my understanding of relational data modeling for structured audit reports, and Next.js App Router helped simplify dynamic routing for shareable audit pages.

Overall, the project strengthened my skills in system design, product thinking, and production-level deployment.
